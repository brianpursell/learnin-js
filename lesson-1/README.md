# Development environment

## Install node

The LTS version should work great.

* https://nodejs.org/en/

## Install yarn

If needed, install homebrew:

`/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`

Then:

`brew install yarn`
